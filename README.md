Sistema de Control de Acceso a Laboratorio

Descripción
Aplicación en Java que permite gestionar el acceso de usuarios a un laboratorio, registrando entradas y salidas, y generando reportes de uso.

Funcionalidades
- Registro de usuarios
- Control de acceso (entrada/salida)
- Validación de accesos
- Historial de accesos
- Cálculo de tiempo total dentro del laboratorio

Tecnologías
- Java
- Archivos .txt
- Arquitectura por capas

Estructura
- Entidades
- AccesoDatos
- LogicaNegocio
- Presentacion

Ejecución
1. Abrir el proyecto en NetBeans
2. Ejecutar la clase Main
3. Usar el menú en consola
